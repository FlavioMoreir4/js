
### Swipe
- HTML
```html
	<script src="//code.jquery.com/jquery-1.12.0.min.js"></script>
	<script type="text/javascript" src="//rawgit.com/flaviomoreiratec/js/master/swipe.js"></script>
```

- Javascript
```javascript
	$('#MyDiv').on('swipeup',function (){
		alert("SwipeUp");
	});
	
	$('#MyDiv').on('swipedown',function (){
		alert("SwipeDown");
	});
```
